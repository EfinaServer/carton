# carton

EfinaServer custom Pelican Panel egg collection.

## Eggs

### Minecraft

| Egg | Description | Link |
|-----|-------------|------|
| LazyMC | Minecraft server proxy that sleeps idle servers and wakes them on player connect. Uses [EfinaServer's fork](https://github.com/EfinaServer/lazymc) with additional fixes and support. | [minecraft/lazymc](minecraft/lazymc/) |

## How to Import an Egg

1. Go to the Pelican Panel admin area
2. Navigate to **Nests** → select or create a Nest
3. Click **Import Egg**
4. Upload the `egg-*.json` file
5. Save and the egg is ready to use
